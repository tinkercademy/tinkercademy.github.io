Graphs

No, they are not the things you draw on graph paper

What is a graph?

We can formally describe a graph as a set of vertices $V$ and a set of edges $E$. Each edge describes a connection between two vertices.

We sometimes also call a vertex a node.

Directed vs undirected graphs

In an undirected graph, there is no distinction between an edge going from $v_0$ to $v_1$ and an edge going from $v_1$ to $v_0$. We can represent them as a set: $\{ v_0, v_1 \}$.

In contrast, a directed graph makes a clear distinction between an edge going from $v_0$ to $v_1$ and an edge going from $v_1$ to $v_0$. If we only have an edge from $v_0$ to $v_1$, we cannot reach $v_1$ from $v_0$. We can represent this as a sequence: $<v_0, v_1>$.

Labelling nodes in graphs

We can label the nodes in our graphs to make them easier to refer to.

In this graph, the nodes are labelled 1 through 6.

Neighbors

Let’s look at node 5. We note that it has 3 neighbors, nodes 4, 2 and 1.

We can also write down a pair wise relationship like such: $(5, 4)$, $(5, 2)$, $(5, 1)$. Alternatively, if you prefer set syntax (seldom used): $\{5, 4\}$, $\{5, 2\}$, $\{5, 1\}$.

Note: Since we are using an undirected graph, there is no distinction between, for example, $(5, 4)$ and $(4, 5)$.

networkx

Creating a graph in networkx

The code below will create an empty graph.

import networkx as nx

G = nx.Graph()

Adding nodes

We can add nodes to the empty graph we created. You can add any type of data which is hashable. You can also add multiple nodes at once by shoving them into a list.

import networkx as nx

G = nx.Graph()

G.add_node(1)
G.add_node("spam")
G.add_node(3.1415926)
# add multiple
G.add_nodes_from([2, 3, 4])

Adding edges

import networkx as nx

G = nx.Graph()
G.add_nodes_from([1, 2, 3, 4])

G.add_edge(1, 2)

e = (2, 3)
G.add_edge(*e) # uses tuple unpacking

# add multiple edges
G.add_edges_from([(1, 3), (1, 4)])

Accessing data in a graph

Gets the neighbors of a particular node.

import networkx as nx

G = nx.Graph()
G.add_nodes_from([1, 2, 3, 4])
G.add_edges_from([(1, 2), (1, 3), (1, 4)])

print(G[1]) 
# alternatively: G.adj[1]

Output: 

{2: {}, 3: {}, 4: {}}

Requesting an adjacency matrix from a networkx graph

import networkx as nx

G = nx.Graph([(1, 2), (1, 3), (2, 3), (2, 4), (3, 4)])

Requesting an adjacency list for a networkx graph

import networkx as nx

G = nx.Graph([(1, 2), (1, 3), (2, 3), (2, 4), (3, 4)])
print(G.adj)
